# s5cmd GitHub Action

Setup [s5cmd](https://github.com/peak/s5cmd) command on GitHub Actions runners.

## Inputs

### `version`

The version of s5cmd to download.

## Example usage

```yaml
uses: derekn/s5cmd-action@v1
with:
  version: 2.0.0
```
